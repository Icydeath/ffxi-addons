Windower アドオン　"where"

このアドオンはsylandro氏作zonenameを参考に作られました。感謝いたします。

■機能
ゲームプレイ中にリージョン・ゾーン名を常時表示させます。
現在場所は左上固定で、リージョン名・ゾーン名の順で表示されます。
デフォルトのフォントは源ノ明朝(Bold)です。ご自身で設定を変更するか、手に入れてください。

lua l where
をinit.txtに追加してください。自動で起動します。

表示がオカシイ場合デフォルトのdataフォルダは削除してみてください。

■コマンド
コマンドは以下。
//where  (or wh)　COMMAND

help ヘルプ表示
jp 日本語表示
en 英語表示
postype 数値 : 0=左上　１＝左下　２＝右上　３＝右下
